Условие

Для функции \(y = f(x)\) заданной табличными значениями в узловых точках \(x_{0} = 1, x_{1} = 2, x_{2} = 3, x_{3} = 4, x_{4} = 5, x_{5} = 6 \) построить интерполяционный полином Ньютона:

  1. При \(y_{0} = -8, y_{1} = -17, y_{2} = -22, y_{3} = -17, y_{4} = 4, y_{5} = 47 \)

  2. Построить интерполяционный полином Ньютона \( N(x) \) , вычислив разделенные разности в таблице вида 3.1.;

  3. Построить график полинома \( N(x) \) интерполирующего функцию \( f(x) \);

  4. Найти (приближенное) значение функции \( f(x) \) в точке \(x=1 \);
Воспользуемся формулой полинома Ньютона:
\(P_{n} (x) = a_{0} + a_{1}(x - x_{0})(x - x_{1}) + ... + a_{n}(x - x_{0})(x - x_{1})...(x - x_{n-1}) \)
где
\(a_{0} = f[x_{0}] \), \(a_{1} = f[x_{1}, x_{0}] \), ..., \(a_{r} = f[x_{r}, x_{r-1}, ..., x_{0}] \)
где
\(f[x_{i}] = y_{i}; \)
\(f[x_{i+1}, x_{i}] = \dfrac{f[x_{i+1}] - f[x_{i}]}{x_{i+1} - x_{i}};...; \)
\(f[x_{r}, x_{r-1}, ..., x_{0}] = \dfrac{f[x_{r}, x_{r-1}, ..., x_{1}] - f[x_{r-1}, ..., x_{0}]}{x_{r} - x_{0}}; \)
